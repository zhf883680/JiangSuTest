# 真假分析

  1. 若A,B (一真一假),则(A要么B)为真[一真一假则真]
  2. ...全真全假...假[全真全假则假]

# 相融选言

# 定义及辨识

  1. 几者中至少1个成立
  2. 逻辑词
    1. 至少一个
    2. 或...或...
    3. 不都不

# 真假分析

  1. 当A,B有真 则(A或B)为真[有真则真]
  2. ...全假....假[全假则假]
  3. -(A或B)=-A且-B
  4. A或B=-A->B=-B->A[前加非,后不变]
