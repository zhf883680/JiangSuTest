# 辨识

逻辑词

  1. 并列词: 且与和
  2. 转折词: 但,却
  3. 递进词: 而且,更

# 真假分析

  1. 当A,B全真时,则A且B为真[全真则真]
  2. ...有假时......假[有假则假]
  3. -(A且B)=-A或-B
